# Fluidstudio_tsl

FluidStudio rebuild using **Three.js r182 WebGPU + native TSL + React Three Fiber**.

- Architecture: `Fluidstudio_tsl/ARCHITECTURE.md`
- Porting/Research notes: `Fluidstudio_tsl/KNOWLEDGE.md`

## Quick Start

```bash
cd Fluidstudio_tsl
npm install
npm run dev
```

## Notes

- WebGPU requires a compatible browser (Chrome/Edge with WebGPU enabled).
- The implementation targets Three.js r182 WebGPU + native TSL patterns (see local examples in `Resource_examples/Threejsr182_webgpuexamples/`).
